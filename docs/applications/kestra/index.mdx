# Kestra

Kestra is an open-source workflow orchestration and scheduling platform for building and running automated data pipelines.

Browse the guides in this section to deploy and build with Kestra in production.

## Available Guides

### [Deploy Kestra](./deploy.mdx)

Deploy Kestra with Docker Compose and a PostgreSQL backend, then run workflows from the web interface.
