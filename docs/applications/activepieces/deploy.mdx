# Deploy Activepieces

Activepieces is an open-source business automation platform that enables you to build no-code and low-code workflows by connecting applications, APIs, and AI services. This production-ready deployment guide walks you through installing Activepieces using Docker Compose with PostgreSQL and Redis on a Linux server.

The complete tutorial is published on **Vultr Docs**.

**→ Read:** [How to Deploy Activepieces – Open-Source Business Automation Platform](https://docs.vultr.com/how-to-deploy-activepieces-open-source-business-automation-platform)

## What You'll Learn

- Deploy Activepieces using Docker Compose
- Configure PostgreSQL and Redis for a production deployment
- Access and complete the initial Activepieces setup
- Validate your deployment and start building automation workflows
