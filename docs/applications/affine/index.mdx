# AFFiNE

AFFiNE is an open-source knowledge management platform that combines documents, whiteboards, databases, and AI-powered collaboration in a single workspace. Whether you're exploring AFFiNE locally or deploying it for production, these guides will help you get started.

## Available Guides

### Deploy AFFiNE

Deploy AFFiNE in a production environment using Docker Compose with PostgreSQL and Redis on a Linux server.

**Published on:** Vultr Docs

**→ Read:** https://docs.vultr.com/how-to-deploy-affine-open-source-knowledge-management-platform



### Install AFFiNE Locally

Install and self-host AFFiNE on Windows using Docker Desktop for development, testing, or personal knowledge management.

**Published on:** freeCodeCamp

**→ Read:** https://www.freecodecamp.org/news/self-host-affine-windows/

## Which Guide Should You Choose?

- **Install AFFiNE Locally** if you want to explore AFFiNE on your Windows computer for learning, testing, or personal use.
- **Deploy AFFiNE** if you're setting up a production-ready instance on a VPS or cloud server using Docker Compose with PostgreSQL and Redis.
