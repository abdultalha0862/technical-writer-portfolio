# How to Self-Host AFFiNE on Windows with WSL and Docker

AFFiNE is an open-source knowledge management platform that combines documents, whiteboards, databases, and AI-powered collaboration into a single workspace. This guide shows you how to self-host AFFiNE on a Windows machine using WSL 2, Docker Desktop, PostgreSQL, and Redis, making it ideal for learning, development, and personal use. :contentReference[oaicite:0]{index=0}

The complete tutorial is published on **freeCodeCamp**.

**→ Read:** [How to Self-Host AFFiNE on Windows with WSL and Docker](https://www.freecodecamp.org/news/self-host-affine-windows/)

## What You'll Learn

- Set up Windows Subsystem for Linux (WSL 2) and Docker Desktop
- Deploy AFFiNE locally using Docker Compose
- Configure PostgreSQL and Redis for local development
- Connect to your self-hosted AFFiNE workspace and manage it on Windows
