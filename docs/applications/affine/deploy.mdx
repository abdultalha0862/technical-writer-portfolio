# How to Deploy AFFiNE – Open-Source Knowledge Management Platform

AFFiNE is an open-source knowledge management platform that combines documents, whiteboards, databases, and AI-powered collaboration into a single workspace. This production-ready deployment guide walks you through deploying AFFiNE using Docker Compose with PostgreSQL and Redis on a Linux server.

The complete tutorial is published on **Vultr Docs**.

**→ Read:** [How to Deploy AFFiNE – Open-Source Knowledge Management Platform](https://docs.vultr.com/how-to-deploy-affine-open-source-knowledge-management-platform)

## What You'll Learn

- Deploy AFFiNE using Docker Compose on a Linux server
- Configure PostgreSQL and Redis for a production-ready deployment
- Access and complete the initial AFFiNE workspace setup
- Verify the deployment and begin organizing documents, whiteboards, and collaborative workspaces
