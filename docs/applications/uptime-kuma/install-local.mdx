# How to Self-Host Your Own Server Monitoring Dashboard Using Uptime Kuma and Docker

Uptime Kuma is an open-source monitoring platform that helps you track the availability of websites, APIs, servers, Docker containers, and other network services from a single dashboard. This guide walks you through self-hosting Uptime Kuma with Docker, configuring your first monitors, and building a reliable server monitoring dashboard.

The complete tutorial is published on **freeCodeCamp**.

**→ Read:** [How to Self-Host Your Own Server Monitoring Dashboard Using Uptime Kuma and Docker](https://www.freecodecamp.org/news/self-host-uptime-kuma-docker/)

## What You'll Learn

- Deploy Uptime Kuma using Docker on a Linux server
- Configure your first uptime monitors for websites, APIs, and services
- Create a centralized monitoring dashboard for your infrastructure
- Monitor the health of your self-hosted applications and receive downtime alerts
