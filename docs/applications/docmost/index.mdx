# Docmost

Docmost is an open-source collaborative documentation and knowledge management platform designed for teams to create, organize, and share documentation in a self-hosted environment. Whether you're deploying Docmost for the first time or managing it in production, these guides cover the complete lifecycle—from deployment and security to monitoring, backups, upgrades, and ongoing maintenance.
