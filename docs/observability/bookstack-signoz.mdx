# Instrument BookStack with OpenTelemetry and SigNoz Using Docker

BookStack is an open-source documentation platform that can be instrumented with OpenTelemetry to collect distributed traces and gain deeper insights into application performance. This guide walks you through deploying BookStack with OpenTelemetry and exporting telemetry data to SigNoz using Docker for end-to-end observability.

The complete tutorial is published on **Dev.to**.

**→ Read:** [Instrument BookStack with OpenTelemetry and SigNoz Using Docker](https://dev.to/abdultalha08/instrument-bookstack-with-opentelemetry-and-signoz-using-docker-2191)

## What You'll Learn

- Deploy BookStack with Docker and OpenTelemetry instrumentation
- Configure the OpenTelemetry Collector and SigNoz
- Collect and visualize distributed traces for BookStack
- Troubleshoot application performance using end-to-end observability
